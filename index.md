## About This Documentation

This portal provides a complete guide to the **Subscrypts Ecosystem**, including:

* Architecture and smart contract specifications
* dApp interaction and integration tutorials
* Discord automation for subscription communities
* Compliance and MiCAR-aligned design philosophy

Each section is crafted to help you understand, build with, or deploy systems with Subscrypts integrated — from conceptual overview to in-depth technical documentation.
